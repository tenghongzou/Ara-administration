## 快速導覽

### 我是新手，從哪裡開始？

1. 閱讀 [專案架構文檔](./01-專案架構文檔.md) 了解系統整體結構
2. 參考 [開發規範指南](./02-開發規範指南.md) 了解程式碼規範
3. 查閱 [部署與建置指南](./05-部署與建置指南.md) 設定開發環境

### 我要開發新元件？

1. 閱讀 [元件設計文檔](./04-元件設計文檔.md) 了解設計系統
2. 參考 [開發規範指南](./02-開發規範指南.md) 的 Svelte 5 規範
3. 查看 `src/lib/components/` 目錄中的現有元件

### 我要整合後端 API？

1. 閱讀 [API 整合規範](./03-API整合規範.md) 了解服務層設計
2. 參考 `src/lib/services/` 目錄中的現有服務

## 專案特色

### 技術亮點

- **Svelte 5 Runes**：使用最新的 Runes API（$state、$derived、$effect）
- **模組化架構**：功能模組獨立、可擴展
- **型別安全**：完整的 TypeScript 覆蓋
- **即時通訊**：WebSocket 即時通知
- **深色模式**：系統主題自動偵測

### 功能模組

| 模組 | 說明 |
|------|------|
| Dashboard | 儀表板統計、快速操作、活動動態 |
| Subscriptions | 訂閱 CRUD、統計分析、行事曆、匯入匯出 |
| Notifications | 通知列表、篩選、批次操作、即時推播 |
| Users | 使用者 CRUD、角色指派、狀態管理 |
| Roles | 角色 CRUD、權限群組管理 |
| Logs | 稽核日誌、全文搜尋、匯出 |
| Account | 個人資料、密碼、2FA、Session 管理 |
| Settings | 系統設定、主題、通知偏好 |
